# 25-26SmartCities
Projet de nichoir connecté basé sur ESP32 TimerCam et Raspberry Pi pour surveiller l’activité des oiseaux. Détection de mouvement via capteur PIR Capture d’images avec caméra OV3660 Transmission des données et images via MQTT Stockage dans base de données MariaDB/MySQL Interface Web (Flask) pour visualiser les photos et l’état du système
